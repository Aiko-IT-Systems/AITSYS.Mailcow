## AITSYS.Mailcow 1.0.0 Documentation
[AITSYS.Mailcow](https://github.com/Aiko-IT-Systems/AITSYS.Mailcow) (AITSYS-MC) is an unofficial .NET wrapper for Mailcow with the goal to be easy-to-use.
